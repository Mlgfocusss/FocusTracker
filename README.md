FocusTracker

A desktop application for habit tracking and task management.
Приложение для отслеживания привычек и задач.

🚀 How to run

Option 1 — Prebuilt version (recommended)
Download the latest release from here:
👉 https://github.com/Mlgfocusss/FocusTracker/releases
Then simply run:
bashappFocusTracker.exe

Option 2 — Build from source
Prerequisites

Qt Creator (Qt 6.9+ recommended)
MSVC (Microsoft Visual C++ Compiler)

Steps

Clone the repository:

bashgit clone https://github.com/Mlgfocusss/FocusTracker.git
cd FocusTracker

Option A: Open in Qt Creator:

Open CMakeLists.txt
Configure the project with MSVC kit
Build and run


🛠️ Technologies

Qt/QML — UI framework
C++ — Core logic
CMake — Build system
MSVC — Compiler
